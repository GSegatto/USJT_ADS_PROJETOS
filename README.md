# USJT_ADS_PROJETOS
Projetos Diversos relacionados a USJT
Este repositório reune arquivos e documentos relacionados a projetos desenvolvidos na faculdade.

## Objetivo
Centralizar e compartilhar tarefas e projetos acadêmicos com professores e colegas, mantendo uma organização clara e acessivel.

## Estrutura
- Documentos (.docs, .pdf, etc.)
- Planilhas (.xlsx, .xlsm)
- Outros materiais de apoio

## Observações
- Este repositorio é de uso acadêmico
- Os arquivos podem ser atualizados conforme novos projetos forem concluídos.
